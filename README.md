# Babel

### Description

The project consists in a client/server architecture voice over IP application, similar to Skype or TeamSpeak written in C++.

Babel is a multi-platform software, it can run on Windows and Linux.

### Installation

Build the project:

`./build.sh`

Run the software.

`./build/bin/babel`
