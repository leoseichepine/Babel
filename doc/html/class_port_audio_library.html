Public function to get the listening callback.

This routine will be called by the PortAudio engine when audio is needed. It may be called at interrupt level on some machines so don't do anything that could mess up the system like calling malloc() or free().

Parameters
inputBufferPointer of the buffer input.
outputBufferPointer of the buffer output.
framesPerBufferThe number of frames per buffer.
timeInfoTiming information for the buffers passed to the stream callback.
statusFlagsFlag bit constants for the statusFlags to PaStreamCallback.
userDataPointer of the data structure.
Returns
the status of the callback.
